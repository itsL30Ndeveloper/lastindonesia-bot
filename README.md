## Getting Started

### This project require NodeJS v14.
This is a Bot Group Type, so you have to enter the bot into a group in order to fully use the command!

# Install Node Modules
```npm i discord.js
npm i cpu-stat
npm i @distube/ytpl

npm i fs
npm i moment
n

```

```node index.js```
